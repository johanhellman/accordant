## üé® Quick Reference

### Primary Colors

- **Primary**: `#2563eb` (Blue)
- **Secondary**: `#7c3aed` (Purple)
- **Gradient**: `linear-gradient(135deg, #2563eb 0%, #7c3aed 100%)`

### Typography

- **Font**: Inter
- **Body**: 16px, Regular (400)
- **Headings**: Bold (700) to Extrabold (800)

### Spacing

- **Base Unit**: 4px
- **Common**: 8px, 16px, 24px, 32px

### Logo

- **Primary**: Sparkles icon + "Accordant" wordmark
- **Icon Only**: Sparkles icon
- **Minimum Size**: 32px height (icon)

## ‚úÖ Brand Compliance Checklist

When creating new materials, ensure:

- [ ] Uses approved logo and follows logo guidelines
- [ ] Colors match brand palette
- [ ] Typography follows type scale
- [ ] Spacing uses 4px base unit
- [ ] Voice and tone match brand guidelines
- [ ] Assets are properly named and organized
- [ ] Accessibility standards are met (WCAG AA)
- [ ] Responsive design principles are followed
